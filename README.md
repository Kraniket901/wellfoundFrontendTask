## To run the project locally

1. Open VS Code > Open New Folder with VS Code
2. Open Terminal (ctrl + ~)
3. git clone https://github.com/Kraniket901/wellfoundFrontendTask.git
4. cd wellfoundFrontendTask
5. npm install
6. npm run start
